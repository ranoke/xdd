```
xxd - make a hexdump or do the reverse.

Usage: xdd.exe [OPTIONS] <FILEPATH>

Arguments:
  <FILEPATH>

Options:
  -c, --cols <COLS>    format <cols> octets per line. Default 16 (-i: 12, -ps: 30, -b: 6). Max 256. [default: 16]
  -u                   use upper case hex letters. Default is lower case.
  -l, --len <LEN>      stop after writing <len> octets.
  -s, --split <SPLIT>  separate every <split> bytes with space. [default: 0]
  -h, --help           Print help
  -V, --version        Print version
```
